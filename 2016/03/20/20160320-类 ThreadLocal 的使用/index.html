类 ThreadLocal 的使用

发表于 2016-03-20 | 分类于 Java Thread |
字数统计: 407 | 阅读时长 ≈ 2

public static 修饰的变量可以让所有线程都共享，但是如果每一个线程都想有自己的共享变量则需要依赖 ThreadLocal 类了。

public static 可以简单理解为所有线程的共享数据，ThreadLocal 变量可以简单理解为单个线程内的共享数据
