Thread 生产者和消费者模式

发表于 2016-03-05 | 分类于 Java Thread |
字数统计: 1,777 | 阅读时长 ≈ 7

生产者和消费者问题描述的是：生产者和消费者在同一时间段内共用同一个存储空间，生产者往存储空间中添加产品，消费者从存储空间中取走产品，当存储空间为空时，消费者阻塞，当存储空间满时，生产者阻塞。

现在使用 wait 和 notify 实现生产者和消费者模式。
