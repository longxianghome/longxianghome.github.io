Spring Framework Beans 组件

发表于 2016-11-18 | 分类于 Spring Framework |
字数统计: 133 | 阅读时长 ≈ 1

Spring Framwork Beans 组件主要在 org.springframework.beans 包下。

在这个包下的所有类主要解决了三件事：Bean 定义、Bean 创建、Bean 解析

其中 Bean 定义、Bean 解析是 Spring Framework 内部实现，使用者只需要关注 Bean 创建即可。

Spring Bean 的创建是典型的工厂模式，顶级接口是 BeanFactory ：

BeanFactory 有三个子类：ListableBeanFactory、HierarchicalBeanFactory、AutowireCapableBeanFactory。

但是上述接口的默认实现类都是 DefaultListableBeanFactory。

中间不同的接口都是区分对象的数据访问所做的限制。
