MyBatis XPath

发表于 2016-07-09 | 分类于 MyBatis 原理 |
字数统计: 152 | 阅读时长 ≈ 1

XML 解析一般有三种方式：

  • DOM 解析
  • SAX 解析
  • StAX 解析

DOM 解析

DOM 解析是基于树形结构，将整个 XML 读入内容构建 DOM 树，比较占内存

SAX 解析

SAX 是基于事件模型的 XML 解析方式，是一种推模式

StAX 解析

StAX 解析与 SAX 类似，不是是拉模式

这里只是做一个简单介绍，不详细展开。

XPath 是 MyBatis 为了方便解析 XML 配置文件专门开发的，XPath 顾名思义可以使用路径来选取 XML 文件中的某个节点或者节点集合
