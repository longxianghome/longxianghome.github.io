线程是多核 CPU 提高利用率的一个利器，赋予操作系统一种可以多任务并行处理的感受。

进程和线程

进程是操作系统基本运行单元

线程是进行中独立运行的子任务

一个进程中至少存在一个线程

Java 默认支持多线程，Java 中实现多线程存在两种方式：

  • 继承 Thread 类
  • 实现 Runnable 接口

Java 中的 Main 方法所在类执行的的称为主线程
