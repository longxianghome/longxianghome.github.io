MyBatis XML 配置文件概述

发表于 2015-09-13 | 分类于 MyBatis |
字数统计: 217 | 阅读时长 ≈ 1

MyBatis XML 配置文件内容比较多，准备按不同的元素分别分析，这里先做一个概述。

MyBatis XML 配置文件层次结构：

configuration 配置

  • properties 属性
  • settings 设置
  • typeAliases 类型别名
  • typeHandlers 类型处理器
  • objectFactory 对象工厂
  • plugins 插件
  • environments 环境
    • environment 环境变量
    • transactionManager 事务管理器
    • dataSource 数据源
  • databaseIdProvider 数据库厂商标识
  • mappers 映射器

上述配置项中只会分析一些常用，不常用的后续有需要再分析

也可以参考官网中的文档说明：http://www.mybatis.org/mybatis-3/
