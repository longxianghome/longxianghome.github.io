链表反转

假设链表有 A –> B –> C –> D 四个节点，若想将链表反转：
1）创建一个新的 Head 节点
2）将 A 节点从原链表中删除，原链表 Head 节点指向 B 节点，并且让新的 Head 节点指向 A 节点
3）将 B 节点从原链表中删除，原链表 Head 节点指向 C 节点，并且让新的 Head 节点指向 B 节点，B 节点指向 A 节点（即在新的链表中从头部插入新结点）
4）将 C 节点从原链表中删除，原链表 Head 节点指向 D 节点，并且让新的 Head 节点指向 C 节点，C 节点指向 B 节点（即在新的链表中从头部插入新结点）
5）将 D 节点从原链表中删除，并且让新的 Head 节点指向 D 节点，D 节点指向 C 节点（即在新的链表中从头部插入新结点）
6）将原链表 Head 指向新链表的第一个节点，完成链表反转
