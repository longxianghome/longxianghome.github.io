Python 变量和简单数据类型

变量的命名规范

变量名只能包含字母、数字和下划线。
变量名可以字母或下划线打头，但不能以数字打头。
变量名不能包含空格，但可以使用下划线来分隔其中的单词。
变量名不能使用关键字和函数名。

字符串
